## **Installation**  
To install PrivyShell, simply run the following command:  

```bash

```


## **Usage**  
Once installed, you can start PrivyShell by running:  

```bash
cd privyshell
python main.py
```

This will connect you to the Discord CLI where you can execute commands interactively.
